Web应用基础课程结业报告

  1. 网站标题：Apex Legends中文官网

  2. 开发过程：

    ​ 本网站是按照Apex Legends英文官方网站进行仿照重建的。

    ​ 为什么选择这款游戏的官网呢？本人是一名游戏爱好者，但作为初学者来讲搭建一个游戏官网首页应该是有很大的难度，更何况没有平面设计的基础。但是Apex的游戏官网有所不同，相比其他网游的平面设计，Apex的官方网站比较简洁，没有各种活动提醒弹窗，也没有各种充值广告，主题也很明确，就是想让你看完了尽快下载——这种简约的网站设计在审美和实用性上都掌握住了玩家的心思。如此超前脱俗的设计理念非常值得学习，也让我有了能够做好的信心。

    ​ 平面设计不是本次作业的主题，所以我选择直接照搬官网的设计思路，本次作业的核心是要学会用Web的语言去重新呈现出目标网页 。

    ​ 即本次网站搭建的过程是一个临摹过程，用目前所学来重新呈现一个网页。

    ​ 所以本网站用了MDB模板以及手写样式（包括JavaScript HTML CSS bootstrap等）。

  3. 遇到问题：

    ​ a. 因为版权问题，网站的美术资源不能直接找到。

    ​ b. 布局会因为页面大小而混乱。

    ​ c. 导航栏粘性失效。

    ​ d. 导航栏下拉到一定程度会被隐藏。

    ​ c. 下拉菜单在粘性导航栏上无法生效。

  4. 如何解决：

    ​ a. 使用爬虫手段以及PS通过直接扒去或间接合成来得到网站美术资源

    ​ b. 使用bootstrap瀑布流式布局。

    ​ c. 父元素不能overflow: hidden或者overflow: auto属性，或者body height:100%
    ​ 必须指定top、bottom、left、right4个值之一，否则只会处于相对定位;父元素

    ​ 的高度不能低于sticky元素的高度

  5. 未解决：

    ​ d. 导航栏下拉到一定程度会被隐藏。

    ​ c. 下拉菜单在粘性导航栏上无法生效。

6.总结：

​ Web这门课程让我们对编写网页产生了浓厚的兴趣，每次上课看着老师给我们展示不同的功能时，就觉得这是一门很神奇的学科。因为这是一门实践性很强的学科，所以课后自己也会对照着老师的PPT，试着完成不同的功能，每次做成了也非常有成就感。Web的前半部分，介绍HTML、CSS、JavaScript的部分相对来说比较简单，而后半部分关于web与数据库连接的内容有点困难，最开始做的时候—直出错。直到后来在老师和同学的帮助下才慢慢解决问题。
​ 由于平时学习比较繁忙，时间比较少，对于网页设计软件的强大功能运用得还不够。在网页设计过程中，由于缺乏训练，所以在设计时也遇到了很多麻烦，有时插入一个特效脚本也是很麻烦。在网页制作过程中，想提高设计水平，想有个理想的网页，必须要勤奋练习，我将在以后的学习过程中不断努力。
